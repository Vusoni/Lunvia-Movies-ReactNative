<h2>Overview</h2>
<p>
A React Native + TypeScript mobile application that consumes <strong>The Movie Database (TMDB) API</strong> to fetch and display movie data. 
Built as part of my React Native learning journey ‚Äî a sandbox to practice API integration, navigation, state management, styled components, and cross-platform mobile UI.
</p>

<hr style="border:1px solid #eee; margin: 20px 0;" />

<h2>Key Features</h2>
<ul>
<li>Browse lists of movies retrieved from TMDB (popular, trending, etc.)</li>
<li>View detailed movie information (title, poster, synopsis, release date, genres, etc.)</li>
<li>Search movies via TMDB API (if implemented)</li>
<li>Cross-platform support (iOS & Android)</li>
<li>Clean code structure with modular components, services and type safety (TypeScript + interfaces)</li>
</ul>

<hr style="border:1px solid #eee; margin: 20px 0;" />

<h2>Tech Stack & Libraries</h2>
<ul>
<li>React Native (TypeScript)</li>
<li>Styling: NativeWind / Tailwind-style configuration via tailwind.config.js</li>
<li>HTTP calls / API integration: services folder (TMDB API wrappers)</li>
<li>Navigation: React Navigation / React Native navigation setup (via app/ structure)</li>
<li>Project configuration: Babel, Metro, tsconfig ‚Äî for type support & React Native build flow</li>
</ul>

<h2>How to Run the App</h2>
<ol>
<li>Download or clone the project repository</li>
<li>Install dependencies with <code>npm install</code></li>
<li>Add your TMDB API key in a <code>.env</code> file</li>
<li>Start the app with <code>npx expo start</code> or <code>react-native start</code></li>
<li>Run on iOS simulator, Android emulator, or Expo Go</li>
</ol>
